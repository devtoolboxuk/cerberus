## Usage

```sh
$ composer require devtoolboxuk/cerberus
```

Then include Composer's generated vendor/autoload.php to enable autoloading:

```php
require 'vendor/autoload.php';
```

```php
use devtoolboxuk\cerberus;

$this->cerberys = new Cerberus();
```
